1 Escribe la diferencia entre una API rest y una API soap

La diferencia es que Rest es un estilo de arquitectura para el diseño de apis a traves del protocolo http y maneja varios formatos de datos como JSON, HTML, XML y la de Api soap es que es un protocolo, un estandar de comunicacion con su conjunto de reglas y solo maneja un formato de dato que es XML

REST=Transferencia de estado presencial SOAP=Protocolo de simple acceso a objectos

3 Escribe la diferencia entre clase, objeto, array, método y función.
  • Una Funcion es un conjuto de codigo y puede estar compuesto por clases,objetos,arrays y otras funciones junto a eso, tiene la forma de agregar parametros para facilitar reutilizar el codigo
  • Una Clase es una coleccion de objectos permitiendo que estos posean atributos y metodos
  • Un Objecto es una coleccion de datos con su determinado valor
  • Un Array es un conjuto de datos y puede contener tantos datos,como objectos los cuales cuentan con metodos nativos para
  • Un Metodo un método es una función que se define dentro de un objeto y se utiliza para realizar una tarea específica en el contexto del objeto

5 Escribe la diferencia entre PHP CGI, fastCGI y FPN
  • PHP es un lenguaje de programacion para crear aplicaciones y sitios webs a diferencia de los demas
  • CGI Interfaz de entrada comun de WWW es un estandar de comunicacion entre un servidor web y un programa en el servidor es una forma comun de generar contenido dinamico
  • FASTCGI Es un protocolo para interconectar servidores web y es una variable de la ya conocida CGI
  • FPN Significa Fully Qualified Pathname, que es el FPN completo del archivo CGI que se está ejecutando "PATH_TRANSLATED", que es el FPN completo del archivo CGI que se está ejecutando.
